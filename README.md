# Expense Tracker

A simple Python console-based expense tracker project made while learning Python fundamentals through the CS50P course.

## Features

- Add expenses
- View all expenses
- Delete expenses
- Monthly budget tracking
- Remaining budget calculation
- Biggest expense detection
- Category-based expense totals
- Error handling for invalid numeric inputs

## Categories

Supported categories:
- Food
- Transport
- Entertainment
- Bills
- Other

Custom categories are also supported.

## Technologies Used

- Python
- Lists
- Dictionaries
- Loops
- Conditionals
- Exception handling (`try/except`)

## What I Learned

This project helped me practice:
- Working with lists and dictionaries
- Building menu systems with loops
- Data accumulation and calculations
- Error handling
- User input validation
- Basic program structure and logic
